TermuCraft is a Minecraft server panel for Termux that aims to make running and managing a server on Android practical instead of painful.

## Status

Early development.

## What It Is

- Web-based control panel for Minecraft servers running in Termux
- Built for Android-hosted server management
- Focused on a simple setup and a usable interface
- Designed as a cleaner, more feature-rich foundation going forward

## Core Features

- Start, stop, restart, and manage a server from the browser
- Live console and server status monitoring
- Server file and property management
- Version download and install flow
- Backups and restore tools
- Player management and admin controls
- Crash recovery and auto-restart support
- Authentication for panel access
- Termux-first workflow instead of desktop-first assumptions

## Goal

TermuCraft is meant to be a practical Minecraft hosting panel for people who want to run a server directly from an Android device through Termux without dealing with a messy manual setup every time.
